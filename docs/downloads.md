## Web Access

To use Liberdus without installation:

1. Go to **[liberdus.com/test](https://liberdus.com/test)**.  
2. The web client loads automatically.  
3. Allow **notifications** for real-time message alerts.  
4. Works offline, messages send automatically when you reconnect.

## Android Installation

1. Open **Google Play Store** and search for **Liberdus**, or use the beta link provided above.  
2. Tap **Join Beta** to become a tester.  
3. Once joined, tap **Install** to download the app.  
4. The app updates automatically via Google Play.
